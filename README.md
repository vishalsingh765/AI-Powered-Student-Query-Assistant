# 🎓 AI-Powered Student Query Assistant

## Overview

The AI-Powered Student Query Assistant is an intelligent chatbot designed to help students with academic and career-related questions. Powered by Generative AI, the assistant provides instant, accurate, and contextual responses on topics such as Programming, Artificial Intelligence, Machine Learning, Career Guidance, and Interview Preparation.

The system aims to act as a virtual mentor, offering personalized assistance and learning support anytime, anywhere.

## ⚙️ How It Works

1. User enters a question through the chat interface.
2. The system processes the query using Generative AI.
3. Relevant context and prompts are generated.
4. The AI model produces a detailed response.
5. Frequently asked queries are cached for faster retrieval.
6. The response is displayed in an interactive chat format.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Powered-Student-Query-Assistant.git

cd AI-Powered-Student-Query-Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure API Key

Create a `config.py` file:

```python
GEMINI_API_KEY = "YOUR_API_KEY"
```

### Run the Application

```bash
streamlit run app.py
```
